# Projeto 4: Downloader de Arquivos Web

- O que ele faz?<br>
  Este script se conecta a uma URL direta de um arquivo (como uma imagem .jpg ou um .pdf) e faz o download desse conteúdo para o computador local.

- Como Usar:

  1- Abra o script downloader.py.

  2- Altere a variável url_do_arquivo para o link direto do arquivo que você quer baixar.

  3- Altere a variável nome_saida para como você quer salvar o arquivo (ex: manual.pdf ou foto.jpg).

  4- Execute o script (python downloader.py). O arquivo será salvo na mesma pasta.
